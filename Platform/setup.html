First-Time Setup

Configure the Agro and More admin for the first time. Run this once, then delete or restrict access to this page.

1

Database schema check

Verifies the schema has been applied to your Supabase project
Before running setup: paste and run Platform/db/schema.sql followed by Platform/db/seed.sql in the Supabase SQL editor ↗. Then come back here and click Check.
2

Enter your Supabase Service Key

Needed once to create auth users. Not stored anywhere.
Security notice: The service key is only used in-browser during this setup session. It is never sent to any third-party server. Delete or password-protect this setup.html file after completing setup.
3

Create demo staff accounts

Creates 4 accounts — MD, Sales Manager, IT Admin, Agronomist
All 4 accounts will be created with password Demo1234!. You can change each password from the Supabase Auth dashboard after setup.
